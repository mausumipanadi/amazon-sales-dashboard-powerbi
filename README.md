# 📊 Amazon Sales Dashboard (Power BI)

This project presents an interactive dashboard built using Power BI to analyze Amazon sales data and extract key business insights.

---

## 🔍 Key Insights

- 💰 Total Sales: 11.58M  
- 📦 Total Orders: 17,845  
- ❌ Cancellation Rate: 4.23%  
- 🧍 Majority revenue comes from B2C customers  
- 👕 Top Category: T-shirts  

---

## 📈 Features

- Sales trend analysis (daily)
- B2B vs B2C customer segmentation
- Top product categories
- State-wise order distribution
- Courier status breakdown

---

## 🛠 Tools Used

- Power BI  
- Excel
